# PROMPT INPUT 1 — Property-Level Data

> **Purpose:** Captures all physical, structural, and immediate financial attributes of a specific property listing.

---

## Instructions

Fill out this data sheet for each property under consideration. Use information from the listing, agent inquiries, and property visits.

> ⚠️ **Physical visits are essential.** Many fields — especially condition ratings, environment assessment (noise, air quality), and systems inspection — cannot be reliably assessed from online listings alone. Always visit the property before completing this form.

---

## Standard Rating Scales

| Scale | Condition Rating | Description |
|-------|------------------|-------------|
| 5 | Excellent | New or like-new, no work needed |
| 4 | Good | Minor wear, cosmetic touch-ups only |
| 3 | Fair | Functional but dated, moderate updates needed |
| 2 | Poor | Significant issues, major renovation required |
| 1 | Critical | Non-functional, full replacement needed |
| 0 | Unknown | Cannot assess / no information |

| Scale | Proximity Rating | Distance |
|-------|------------------|----------|
| 5 | Immediate | < 500m / < 5 min walk |
| 4 | Very Close | 500m - 1km / 5-15 min walk |
| 3 | Close | 1-2km / 15-30 min walk |
| 2 | Moderate | 2-5km / requires transport |
| 1 | Far | > 5km |

| Scale | Risk/Nuisance Level |
|-------|---------------------|
| 1 | Low / Minimal |
| 2 | Moderate / Acceptable |
| 3 | High / Concerning |
